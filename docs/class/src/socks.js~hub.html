    public constuctor(opts: Object) source

    hub constructor, sets up config for backoff, WebSocket creation, internal message buffer, does some light sanity checking.

    Params:

    NameTypeAttributeDescription
    opts Object
    opts.url string
    • nullable: false

    Server endpoint hosting websocket
    opts.subscribePath string
    • optional
    • default: '/api/join'

    Server side message endpoint that services channel subscriptions.
    opts.factor number
    • optional
    • default: 200

    Backoff step factor ("to be increased by")
    opts.max number
    • optional
    • default: 2000

    Ceiling for backoff wait time

    public makeSocket() source

    Create a new websocket and hold the ref to it internally. Also decorates the new WebSocket with a write method to serialized outgoing messages.

    public request(req: object): Promise source

    Write a message to the server expecting a response to come back keyed with the same requestId as aformentioned write.

    Params:

    NameTypeAttributeDescription
    req object

    Request object to be written to WebSocket. Will be deocorated with uuid.v1 and pending = true.

    Return:

    Promise

    Rejects if error property present on payload from server, resolves otherwise.

    public subscribe(channel: string, topic: string, cb: function(msgData: Object)) source

    Initiate write to WebSocket to communicate interest in subscribing to messages across a given channel. Pass subscribePath as option to constructor to configure "endpoint" used to initiate subscription on server.

    Params:

    NameTypeAttributeDescription
    channel string

    Name of channel, to be send on body.channel property of outgoing message.
    topic string

    Optional topic in channel, could be used as a sort of subcategory of interest.
    cb function(msgData: Object)

    Callback to be invoked on each message written to channel by server.

    public unsubscribe(channel: string, cb: function) source

    Send message to server indicating desire to unsubscribe from given channel.

    Params:

    NameTypeAttributeDescription
    channel string

    Name of channel to unsub from.
    cb function

    Reference to the callback originally passed on subscription creation.
